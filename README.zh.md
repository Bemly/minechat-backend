| [server/](server/) | Rails + Phlex | Web 前端 + Marshal API |
| [tui/](tui/) | cli-ui | 终端界面客户端 |
| [gui/](gui/) | Gosu | 桌面 2D GUI 客户端 |
| [android/](android/) | Ruboto | Android 应用 |
